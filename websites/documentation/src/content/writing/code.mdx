# Code blocks

You can use standard Markdown syntax for code blocks. It is also possible to render code blocks in two different themes and add titles to them.

- To add titles for the code blocks, pass `title` in the syntax.
- The primary theme is the default. To render the code block in the secondary theme, pass `secondaryTheme` in the syntax.

**Primary theme (default)**

<SideBySide>

~~~
```md title="Code block in primary theme"
Sample code goes here.
```
~~~

```md title="Code block in primary theme"
Sample code goes here.
```

</SideBySide>

```

```

</SideBySide>

## With highlighted lines

It is also possible to highlight specific lines in a code. To highlight, pass `highlightLines` in the syntax and enter the range or specific line to be highlighted.
