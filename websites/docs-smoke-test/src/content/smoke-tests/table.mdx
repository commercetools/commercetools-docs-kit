# Generic styles for tables

- Table is always left aligned in its container
- Row colors alternate between white and grey
- Columns can be left, center or right aligned
- Cell padding is 8px
- Header row border top and bottom is 2px thick.

| Left Aligned  | Center Aligned | Right Aligned |
| ------------- | :------------: | ------------: |
| Hello         |     short      |        \$1600 |
| World         |    content     |          \$12 |
| zebra stripes | avoid insects  |           \$1 |

# Table width and cell content alignment

- Max width of table is never larger than its parent's
- Cell contents are always center aligned on the vertical axis.

| Left Aligned  | Center Aligned | Right Aligned | This right aligned content is a long content that extends to the edge of the screen |
| ------------- | :------------: | ------------: | ----------------------------------------------------------------------------------: |
| Centered cell |    Content     |       Content |                                                                             Content |
| Content       |    Content     |       Content |                                                                             Content |
| Content       |    Content     |  Centred cell |                                                                             Content |
