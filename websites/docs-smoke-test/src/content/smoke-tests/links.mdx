# Links

## Common use cases

### Link pointing to an empty string

```md
[]()
```

| Link  | Type           | Expected outcome                                |
| ----- | -------------- | ----------------------------------------------- |
|  []() |  `anchor-link` |  It should render as a link that does not work. |

### Link pointing to just whitespace-like

```md
[ ]()
```

| Link   | Type           | Expected outcome                                |
| ------ | -------------- | ----------------------------------------------- |
|  [ ]() |  `anchor-link` |  It should render as a link that does not work. |

### Link pointing to a different website

```md
[Link](https://commercetools.com)
```

| Link                               | Type             | Expected outcome                                                         |
| ---------------------------------- | ---------------- | ------------------------------------------------------------------------ |
|  [Link](https://commercetools.com) |  `external-link` |  It should work and be an OutboundLink that tracks via google analytics. |

|  [Link](#last-heading) |  `anchor-link` |  It should jump to that position without visible scrolling or reload. |
