The GatsbyJS plugin supports passing extra properties in a custom notation, so that could be for example a title.

```javascript{title=A javascript example}
// Constructor
var anObject = new Object();

// Object literal
var objectA = {};
var objectA2 = {};
var objectB = { index1: 'value 1', index2: 'value 2' };
```

Syntax: (not supported by syntax highlighting in VSCode)

```
   javascript{title=A javascript example}
```
